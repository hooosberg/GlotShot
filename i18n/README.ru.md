## 🚀 Введение

**GlotShot** — это мощный инструмент, разработанный для разработчиков и дизайнеров мобильных приложений, позволяющий без усилий создавать потрясающие изображения предпросмотра (скриншоты) и иконки для **App Store** и **Google Play**.

Благодаря встроенной поддержке **локализации** и **пакетной обработки**, GlotShot помогает расширить глобальный охват вашего приложения, создавая профессиональные маркетинговые материалы за считанные минуты.

## ✨ Особенности

- **🎨 Двойные режимы дизайна**: Легко переключайтесь между **Дизайном постеров** для скриншотов и **Дизайном иконок** для иконок приложений.
- **🌍 Мультиплатформенная поддержка**: Создавайте материалы, соответствующие стандартам **iOS (App Store)**, **Android (Google Play)**, **macOS** и **Windows**.
- **⚡ Пакетная обработка**: Создавайте скриншоты для нескольких устройств и языков за один раз. Экономьте часы ручной работы.
- **🤖 ИИ переводы**: Интеграция с **Ollama** для локальных, конфиденциальных переводов (без отправки данных) ваших маркетинговых текстов на несколько языков.
- **🖼️ Фабрика иконок**: Автоматическая обрезка и экспорт иконок приложений для всех основных платформ из одного исходного изображения.
- **💎 Современный UI**: Элегантный интерфейс в темном режиме с эффектом матового стекла, разработанный в стиле macOS.
- **📦 Умный экспорт**: Экспортируйте ваши активы, организованные по платформе и языку, готовые к загрузке.

## 📥 Установка

### macOS
1. **[Download from the Mac App Store](https://apps.apple.com/us/app/glotshot-screenshot-maker/id6757913340?mt=12)** — always the latest version.

Or manually via GitHub:

1. Перейдите на страницу [Релизов](https://github.com/hooosberg/GlotShot/releases/latest).
2. Скачайте последнюю версию файла `.dmg`.
3. Откройте образ диска и перетащите **GlotShot** в папку «Программы».
