## 🚀 简介

**万语图** 是一款专为移动应用开发者和设计师打造的强大工具，旨在轻松制作精美的 **App Store**、**Google Play**、**Steam** 以及 **Windows Store** 预览图（截图）及应用图标。

**万语图** 内置**本地化**和**批量处理**支持，帮助您在几分钟内生成专业的营销素材，从而拓展应用的全球市场影响力。

## ✨ 功能特点

- **🎨 双重设计模式**：在**海报设计**（应用截图）和**图标设计**（Logo 制作）之间无缝切换。
- **🌍 多平台支持**：一键生成符合 **iOS (App Store)**、**Android (Google Play)**、**macOS**、**Windows Store** 以及 **Steam** 标准的素材。
- **⚡ 超强批量处理**：一次性为多种设备和语言制作截图，节省数小时的重复劳动。
- **🤖 AI 智能翻译**：集成 **Ollama**，支持本地、隐私安全的营销文案多语言翻译，告别生硬机翻。
- **🖼️ 图标工场**：只需一张源图片，即可自动识别主体，智能裁剪并导出所有主流平台的应用图标。
- **📏 智能尺寸推荐**：不懂苹果复杂的尺寸要求？没关系，万语图内置了 App Store 审核通过率最高的尺寸组合，一键生成 6.9" 和 5.5" 黄金搭档。
- **📦 智能导出**：按平台和语言自动整理导出文件，直接可用于商店上传。

## 📥 安装指南

### macOS 用户
1. 访问 [Releases](https://github.com/hooosberg/GlotShot/releases/latest) 页面。
2. 下载最新的 `.dmg` 安装包。
3. 打开安装包并将 **万语图** 拖入应用程序文件夹即可。

### Windows 用户
1. 访问 [Releases](https://github.com/hooosberg/GlotShot/releases/latest) 页面。
2. 下载最新的 `.exe` 安装程序。
3. 运行安装程序即可完成安装。

### Linux 用户
1. 访问 [Releases](https://github.com/hooosberg/GlotShot/releases/latest) 页面。
2. 下载 `.AppImage` 或 `.deb` 文件。
3. 对于 `.AppImage`文件，请先赋予执行权限 (`chmod +x GlotShot-*.AppImage`) 然后运行。

## 💻 系统配置要求

| 操作系统 | 最低要求 | 推荐配置 |
|----|----------------------|-------------|
| **macOS** | macOS 11.0 (Big Sur) 或更高 | Apple Silicon (M1/M2/M3) |
| **Windows** | Windows 10 (64位) | Windows 11 |
| **Linux** | Ubuntu 20.04+ (或兼容发行版) | 最新 LTS 版本 |

- **内存**: 至少 4GB，推荐 8GB。
- **存储空间**: 预留 500MB 可用空间。

## 🛠️ 开发构建

**万语图** 基于 **Electron**、**React** 和 **Vite** 构建。

### 前置要求

- Node.js (v18 或更高版本)
- npm 或 pnpm

###以此开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/hooosberg/GlotShot.git
   ```
2. 安装依赖：
   ```bash
   cd GlotShot
   npm install
   ```
3. 运行开发模式：
   ```bash
   npm run dev
   ```
4. 构建生产版本：
   ```bash
   npm run build
   npm run electron:build
   ```
