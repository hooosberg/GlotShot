## 🚀 はじめに

**GlotShot** は、モバイルアプリ開発者やデザイナーが **App Store** や **Google Play** のプレビュー画像（スクリーンショット）やアイコンを簡単に作成できるように設計された強力なツールです。

**ローカリゼーション**と**バッチ処理**のサポートが組み込まれており、プロフェッショナルなマーケティング素材を数分で生成して、アプリのグローバル展開を支援します。

## ✨ 特徴

- **🎨 デュアルデザインモード**: スクリーンショット用の**ポスターデザイン**とアプリアイコン用の**アイコンデザイン**をシームレスに切り替えられます。
- **🌍 マルチプラットフォーム対応**: **iOS (App Store)**、**Android (Google Play)**、**macOS**、**Windows** の基準に準拠した素材を生成します。
- **⚡ バッチ処理**: 複数のデバイスと言語のスクリーンショットを一度に作成します。手作業の時間を大幅に削減します。
- **🤖 AI翻訳**: **Ollama** と統合し、マーケティングコピーをローカルでプライバシーを守りながら多言語に翻訳します。
- **🖼️ アイコンファブリック**: 1つのソース画像から主要なすべてのプラットフォーム用のアプリアイコンを自動的に切り抜き、エクスポートします。
- **💎 モダンUI**: macOSのネイティブな操作感を目指して設計された、すりガラス効果を備えた洗練されたダークモードインターフェース。
- **📦 スマートエクスポート**: プラットフォームと言語ごとに整理してエクスポートし、すぐにアップロードできる状態にします。

## 📥 インストール

### macOS
1. [Releases](https://github.com/hooosberg/GlotShot/releases/latest) ページにアクセスしてください。
2. 最新の `.dmg` ファイルをダウンロードします。
3. ディスクイメージを開き、**GlotShot** をアプリケーションフォルダにドラッグしてください。
